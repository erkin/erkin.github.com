cond-expand

← Back up

https://srfi.schemers.org/srfi-0/srfi-0.html is the first https://srfi.schemers.org/, describing a simple macro cond-expand that expands code at macro expansion time, based on symbols/keywords that stand for features the implementation chooses to provide as a guarantee of portability, (not unlike Common Lisp's #+ read macro). To give an example;

(display (cond-expand ((and srfi-1 srfi-8) "This implementation contains the list library and the `receive' procedure.") ((or guile chicken) "This is either Guile or CHICKEN Scheme.") (else "This implementation is neither CHICKEN Scheme nor Guile, nor does it support SRFI-1 and SRFI-8."))) (newline)
